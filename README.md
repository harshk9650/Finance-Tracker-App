## 🌟 Overview

**Finance Tracker App** is a modern, full-featured personal finance management web application. It helps users track income and expenses, set budgets, visualise spending patterns, and achieve savings goals – all with a beautiful, responsive interface.

Built as a production-ready project for a Frontend Intern portfolio, it demonstrates:

- ✅ Authentication & protected routes
- ✅ Transaction CRUD with search/filter
- ✅ Interactive analytics (charts)
- ✅ Budget planner with alerts
- ✅ Dark/light mode & currency switcher
- ✅ CSV export & report download
- ✅ SEO optimisation & performance best practices

---

## 🔍 Features

### 🔐 Authentication
- Secure login / signup with local storage
- Protected dashboard – only authenticated users can access data

### 📊 Dashboard
- Total balance, income, expenses, monthly savings cards
- Recent transactions list
- Budget progress bar with warning alerts

### ➕ Add Transaction
- Choose income or expense
- Category selection (Food, Transport, Bills, Salary, etc.)
- Amount, date picker, optional notes

### 📜 Transaction History
- Search by title/notes
- Filter by category, type (income/expense), and date range
- Edit or delete any transaction

### 📈 Analytics Section
- Pie chart – expense breakdown by category
- Bar chart – monthly income vs expenses
- Line chart – income/expense trends over time
- Download full financial report (JSON)

### 🎯 Budget Planner
- Set monthly budget
- Visual progress bar
- Warning when spending exceeds 80%
- Alert when budget is exceeded

### 🎨 Premium Features
- **Dark / Light mode** – toggle theme
- **Currency switcher** – USD, EUR, GBP, INR, JPY, CAD, AUD
- **Export transactions** to CSV
- **Download reports** (JSON summary)
- **Notification alerts** (toast messages)

### 📱 Responsive UI
- Mobile-first design
- Tablet & desktop optimised
- Smooth animations with Framer Motion

### ⚡ Performance & SEO
- Lazy loading components
- Optimised images
- Fast page loads (good Core Web Vitals)
- Meta titles, descriptions, Open Graph tags
- Semantic HTML structure
